FAP: Geral, Equipe do FAP, Turmas de frontend e backend isso inclui canais de texto e voz, residência e os arquivados.
Lista de Tarefas: Tarefas que são designadas para os bolsistas pelo coordenador ou por eles mesmos.
Bolsitas: um canal onde os bolsistas de diferentes projetos podem se encontrar e também há um lista de tarefas.
EMM: canais voltados para o desenvolvimento do EMM, onde há frontend, backend, hardware, statioms e lista de tarefas.
ADM: Acompanhamento dos alunos do FAP, secreataria, Assuntos gerais relacionados ao FAP na parte administrativa e o ADM.


////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////

O servidor Softex Labs é uma plataforma robusta e organizada que suporta diversas atividades dentro da Softex. Com mais de 20 categorias e mais de 150 canais disponíveis, além de 111 cargos específicos, o servidor é projetado para facilitar a comunicação e colaboração entre os membros da comunidade.

Entre as categorias principais e mais ativas estão:

Lobby: Canal dedicado à verificação de novos usuários, boas-vindas e registro de atividades de entrada e saída.

Softex Labs: Foco em divulgações de eventos como webinars, palestras e oficinas.

Formação Acelerada em Programação (FAP): Inclui canais gerais, equipes, turmas de frontend e backend, residência e arquivos relacionados.

Lista de Tarefas: Gerenciamento das tarefas atribuídas aos bolsistas pela coordenação ou pelos próprios membros.

Bolsistas: Comunidade para bolsistas de diferentes projetos, com espaço dedicado para interação e gestão de tarefas.

Estação de Monitoramento Modular (EMM): Focado no desenvolvimento de frontend, backend, hardware, estações e gestão de tarefas específicas.

ADM: Gestão administrativa relacionada ao acompanhamento dos alunos, secretaria e assuntos gerais administrativos do FAP.


///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////

A hierarquia de cargos no Discord Softex Labs se dá dessa forma :

Admin:

Manager: São os responsáveis pelo gerenciamento do Projeto Softex Labs podem ver, gerenciar canais, cargos, criar convites, criar e alterar apelidos, expulsar membros, mandar mensagens.

Discorder: tem a permissão de administrador, pois são responsáveis pelo desenvolvimento de Bots Funcionais dentro do próprio servidor.

Bolsistas SoftexLabs : são bolsistas que fazem parte do programa, podem ver os canais, mas não podem gerencia-los, ficam apenas com principais funções de comunicação mas não pode gerenciar um servidor.

O mesmo vale para os cargos, Bolsistas Robótica e IOT.


FAP RECIFE: são cargos que são dados para os professores da Formação Acelerada em Programação.

EMM: Cargo esse dado para os bolsistas que também fazem parte da Estação de monitoramento modular, este cargo te da as mesmas permissões que os bolsitas softexlabs, porem dentro da categoria EMM.

Fap Equipe: esta cargo é designado para a equipe que compõe o FAP, tanto em questões secretárias como administrativas, psicológicas e pedagógicas.

Residente: cargo dado para alunos que conseguiram fazer parte do programa de residência do FAP.

Turmas: este cargo é dado para os alunos que fazem parte de alguma turma do Fap, seja frontend ou backend.

Recém-chegado: este cargo é dado para a pessoa que acessa o softex labs pela primeira vez e ainda não fez a verificação para receber os cargos apropriados.

Bot: o cargo de bot é dado apenas para os bots, isso inclui os próprios bots do discord como também os bots desenvolvidos pela equipe de desenvolvimento de Bots funcionais.
