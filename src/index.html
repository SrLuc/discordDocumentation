Documentation template with Bootstrap 5 and Vite.js

Introduction

This is a free documentation template with a modern sleek design, built on Bootstrap 5 and Vite.js. Just download it and easily customize for your needs.

See how to get started

Featuring

Semantic html5 code

The source code is valid html5 and it is well organized.

Dynamic sidebar menu

The sidebar navigation is dynamically collapsing with scrollspy.

Mobile friendly layout

The template is 100% responsive, it looks great on every device.

Sketch file included

The design source sketch file is bundled with the template.

Resources

The following free resources were used to complete this project:

  • Bootstrap - the latest version, Bootstrap 5 gives the base structure of the template
  • Vite.js - a new generation frontend developer tooling
  • Sass - the most popular css preprocessor to write advanced styles
  • GitHub - version controller tool, used to maintain the updates of the project
  • Unicons - a free collection of scaleable and colorable svg UI icons
  • Prism.js - a javascript library for syntax highlighting code blocks

Available tags

You can use these html5 tags to define your layout base:

Tag Description
html Represents the root of an HTML document.
article It specifies independent, self-contained content.
section It defines a section in a document, similar as a content div.
header It represents a container for introductory content or a set of navigational links.
footer This tag defines a footer for a document or section.
