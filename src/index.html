    Softex Labs

    Antes de começar, é importante saber que o servidor Discord Softex Labs está organizado em duas partes principais: Categorias e Canais.

    Categorias

  • São o nível mais alto na estrutura de organização do servidor Discord.
  • Agrupam e organizam os canais relacionados dentro de um tema específico.
  • Permitem uma divisão e facilitam a navegação para os membros do servidor.

    • Canais

  • São encontrados dentro de categorias e são onde ocorrem as interações.
  • Podem ser textuais ou de voz e vídeo.
  • Cada canal tem um propósito e também suas próprias regras e configurações.

    • Estrutura do Servidor

    O servidor Softex Labs é uma plataforma robusta e organizada que suporta diversas atividades dentro da Softex. Com mais de 20 categorias e mais de 150 canais disponíveis além de 111 cargos específicos, o servidor é projetado para facilitar a comunicação e colaboração entre os membros da comunidade.


    Nosso sistema de canais é organizado de maneira a facilitar o acesso às informações e recursos relevantes, promovendo uma experiência eficiente e intuitiva para todos os usuários.


    Em FAP, encontramos canais para discussões gerais, comunicação da equipe do FAP, turmas de frontend e backend (com canais de texto e voz), informações sobre residência, e arquivos de tópicos concluídos ou inativos. Esta categoria organiza todas as interações e informações relacionadas ao FAP.

    No Lobby, novos usuários passam pela verificação, recebem mensagens de boas-vindas e podem visualizar os logs de entrada e saída do servidor. Este espaço garante uma introdução organizada e acolhedora para todos os membros.

    EMM contém canais voltados para o desenvolvimento da Estação de Monitoramento Modular, incluindo subcanais para frontend, backend, hardware, stations e uma lista de tarefas. Esta categoria organiza detalhadamente todas as áreas do projeto EMM.

    A categoria Softex Labs é dedicada às divulgações de eventos, webinars, palestras e oficinas, mantendo todos informados sobre as principais atividades e oportunidades de aprendizado.

    A Lista de Tarefas é onde são designadas e organizadas as tarefas dos bolsistas, seja pelo coordenador ou pelos próprios membros, permitindo um acompanhamento claro do progresso das atividades.

    Na categoria Bolsistas, bolsistas de diferentes projetos podem se encontrar e colaborar, além de acessar uma lista de tarefas específica para suas atividades, facilitando a coordenação e execução dos projetos.

    Por fim, ADM abrange o acompanhamento dos alunos do FAP, comunicação com a secretaria, e discussões administrativas gerais, garantindo uma gestão eficiente e organizada do FAP.

    The following free resources were used along with this project:

    • Bootstrap - the latest version, Bootstrap 5 gives the base structure of the template

    • Prism.js - a javascript library for syntax highlighting code blocks

    Dependencies

    There are a few npm dependencies need to be installed before serving the project:

    • popperjs - a positioning solution for tooltips and popovers
    • autoprefixer - a postcss plugin that adds vendor prefixes to the output css files
    • bootstrap - latest version of Bootstrap, what is a powerful and extensible frontend toolkit
    • postcss - a tool for transforming CSS with JavaScript (required for using autoprefixer)

    Note: to be able to install these packages via npm, you need to have Node.js installed on your system.
